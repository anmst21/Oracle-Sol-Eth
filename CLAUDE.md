# Project Context

## TSL Reference Projects
The following sibling projects contain working TSL (Three.js Shading Language) examples to use as reference:

- `../tsl/tsl-ascii/` — ASCII shader post-processing on a 3D scene
  - Key files: `src/app.js`, `src/getMaterial.js`
- `../tsl/tsl-vite/` — Canvas drawing on mouse move with image rendering
  - Key files: `src/app.js`, `src/getMaterial.js`

When implementing Three.js/TSL features, read these files for patterns and working examples.
